Link your LeetCode account

Shows a read-only "solved on LeetCode" mark next to matching items -- it never checks/unchecks your own progress, that's always manual.

Quick sync uses a community proxy (alfa-leetcode-api) and only sees your ~20 most recent accepted submissions -- LeetCode's own API doesn't expose full solve history without authentication, and no public proxy supports that.

Full sync (your entire solve history, needs a one-time free setup)

This needs a tiny proxy you deploy yourself on Cloudflare's free tier (~5 min, no card needed) -- the code and step-by-step instructions are in cloudflare-worker/leetcode-proxy.js in the repo. It exists because your session cookie can only ever be attached to a LeetCode request by a server, never by JavaScript running in this page.

Treat this cookie like a password -- it's stored only in this browser's local storage, sent only to the Worker URL above, and never touches any server of mine. Log out of LeetCode (or rotate the cookie) any time to revoke it.

Progress saves automatically to this browser, and to your account when signed in.
